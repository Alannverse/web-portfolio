About Me

Saya adalah mahasiswa Teknik Informatika dengan minat besar dalam pengembangan web. Saat ini, saya aktif membangun portofolio pribadi serta mengembangkan berbagai proyek web, termasuk untuk institusi pendidikan. Proyek-proyek ini bertujuan untuk menampilkan keterampilan saya sekaligus memperdalam pemahaman tentang berbagai framework yang relevan dengan pengembangan web. Melalui eksplorasi dan praktik ini, saya berkomitmen untuk terus memperluas wawasan serta meningkatkan kemampuan teknis dalam dunia pengembangan web.

Selain itu, saya juga tertarik untuk terus belajar tentang teknologi terbaru. Saya percaya bahwa dengan terus belajar dan beradaptasi, saya dapat memberikan kontribusi yang lebih baik di dunia digital.
